## 什么是CookUnity？

CookUnity是美国第一家"大厨对消费者"的送餐服务平台，2015年在布鲁克林成立。这个平台不是让你自己做饭，也不是送你半成品食材，而是直接把大厨做好的完整餐食送到你家门口。

说白了，就是把餐厅大厨请到你家冰箱里，但价格比去餐厅吃便宜多了。

平台上有70多位专业大厨，很多都是获奖无数的名厨，像Jose Garces、Esther Choi这些Food Network的明星厨师。每周菜单都会轮换，有200多道菜可以选，从美式家常菜到亚洲风味，从素食到高蛋白餐，应有尽有。

最关键的是，这些餐食都是新鲜配送，不是冷冻食品。每个餐盒上都标注了营养信息、加热说明，还有厨师的照片和签名。微波炉热2-3分钟，或者用烤箱热10-15分钟，就能吃上一顿餐厅水准的饭。

我自己试了一周，选了8道菜，从亚洲风味到美式家常都有。

2. **地中海烤三文鱼** - 三文鱼烤得恰到好处，外面有点焦香，里面还保持着嫩度。搭配的蔬菜也很新鲜。我本来担心鱼肉类重新加热会变干，但完全没有这个问题。

3. **墨西哥牛肉Tinga塔可** - 牛肉用墨西哥传统香料腌制，味道很正宗。塔可饼皮也不会太硬。

4. **亚洲风味芝麻姜汁豆腐Poke碗** - 素食选项里我最喜欢的。豆腐煎得外脆内嫩，配上新鲜蔬菜和米饭，口感层次丰富。

**加热效果：**

微波炉热2-3分钟是最快的，但我更推荐用烤箱。烤箱加热大概需要10-15分钟（看菜品），但口感明显更好，特别是有酱汁的菜，烤箱能让酱汁重新焕发香气。

每个餐盒上都有两种加热方式的说明："快速版"（微波炉）和"大厨版"（烤箱或炉灶）。如果时间充裕，强烈建议选大厨版。

## 和其他送餐服务比起来怎么样

市面上送餐服务很多，CookUnity和它们比起来各有特点。

**VS HelloFresh/Blue Apron（食材配送）：**
CookUnity完胜。HelloFresh那些是给你食材和菜谱，你还得自己做。CookUnity是做好的，直接热就行。如果你就是想省时间，根本没得比。

**VS Factor（预制餐）：**
Factor也是预制餐，但菜单选择少很多（每周只有45道左右），而且口味比较偏重健康餐，没那么多元化。CookUnity有200多道菜，从健康餐到重口味都有。不过Factor的份量控制得比较标准，适合严格控制饮食的人。

**VS Trifecta（运动员餐）：**
如果你是专业健身人士或运动员，Trifecta可能更合适。他们的Performance系列蛋白质能达到52克，而且有详细的宏量营养素追踪。CookUnity虽然也有高蛋白选项，但更偏美食导向，不是那么硬核的健身餐。

**VS 自己做饭：**
这个就看个人了。自己做饭肯定更省钱，但要花时间买菜、做饭、洗碗。如果你时间紧张或者不喜欢做饭，👉 [CookUnity的便利性](https://www.cookunity.com)还是很值得的。而且从单价来看，其实和去餐厅吃或叫外卖差不多，但质量更稳定。

## 配送和取消政策

**配送范围：**
CookUnity目前覆盖美国大部分地区，主要城市都能送到。具体包括纽约、洛杉矶、芝加哥、奥斯汀、西雅图、亚特兰大、迈阿密等。农村地区可能覆盖不到，注册时可以输入邮编查看是否能配送。

**配送时间：**
你可以自己选择配送日。餐食会在你选定的日子送到，一般是早上到晚上之间。他们会发短信提醒你餐食快到了。

有用户反映过配送时间可能比较晚，甚至晚上9点才到，但好在会提前发短信通知，不至于让餐食在门外放太久。

**取消政策：**
这个要注意时间。CookUnity是每周扣费的，通常配送日前4-5天是截止时间。比如你周一配送，那周三或周四之前要取消，不然还会扣这周的费用。

取消过程很简单，在账户里点几下就行，没有电话客服骚扰你。不过如果已经过了截止时间，那周的餐还是会照常扣费和配送。

## 环保和可持续性

CookUnity在环保方面做得还不错。

餐盒使用的是可工业堆肥的材料，如果你当地有工业堆肥设施，可以把餐盒送过去。没有的话，清洗后也可以和其他纸制品一起回收。

他们还在推广可重复使用包装计划，配送时用的保温袋和冰袋都可以重复使用或回收。

食材方面，CookUnity强调直接和负责任的供应商合作，尽量减少食物浪费和碳排放。每个大厨都在当地厨房小批量制作餐食，比大规模工业化生产更环保。
